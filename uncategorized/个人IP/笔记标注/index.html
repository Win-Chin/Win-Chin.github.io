笔记符号

  • 将笔记符号分为 行内标记、页边距标 记两种
    行内标记
  • 下划线 “ _____ ”：一般句子标识
  • 波浪线 “ ~~~~ ”：重点句子标识
  • 矩形框“ □ ”：框住一般性词语
  • 圆圈“ ⭕️ ”：圈出关键性词语
  • 斜线“ / ”：标明段内层次
  • eg: 一鼓 / 作气，再 / 而衰，三 / 而竭。
    行外标记

缩略语：
eg.举例
etc. 等等et. cetera
S:开始start
E:结束end
Q:问题question
A:答案answer
P:计划plan
I:点子idea
T:行程trip
W:工作work
L:生活life

______ 一般重点句子
△△△ 一般性词语
□ 重点词语
/  段内层次
○ 关键性词语
2. 页边距标记
☆ 重点条目
△ 一般重要条目
•  关键字
？ 质疑
○ 待解决条目
!  注意
○√  已解决条目
☆!   犯过错的地方
2. 数字符号
↑增加  ↓减少  ∵因为  ∴所以  ＞大于
＜小于  ＝等于  ＆与,对比       ︱或   
≈约等  ≠不等
